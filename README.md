# Hfut-studyroom

合肥工业大学宣城校区自习室预约系统

## 参与开发

本项目使用 Taro + Nerv + TaroUI 构建

### 安装依赖

```shell
git clone https://github.com/HFUT-Newx/hfut-studyroom.git
cd hfut-studyroom/
yarn install
```

### 调试

```shell
yarn dev:weapp
yarn dev:h5
```

### 编译

```shell
yarn build weapp
yarb build h5
```

### 部署和发布

暂无
